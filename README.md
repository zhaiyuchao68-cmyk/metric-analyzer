# 指标拆解分析工具

为呼叫中心业务提供自动化的指标拆解分析。

## 快速开始

```bash
pip install -r requirements.txt
streamlit run app/app.py
```

## 支持的拆解方法

- **加法拆解**：绝对量指标按维度拆分
- **减法拆解**：衍生差值指标
- **乘法拆解**：流程链路指标（LMDI方法）
- **除法拆解**：效率/比例指标
- **双因素拆解**：相对数指标按维度拆分

## 预设指标

客户满意度、接通率、首次解决率、平均处理时长、单位人员效率等11个呼叫中心常见指标。

## 自定义指标

复制 `custom_metrics.yaml.example` 为 `custom_metrics.yaml`，按格式填写后导入。

## 数据格式

上传 Excel/CSV 文件，需包含：
- 拆解维度列（如语言线、班组）
- 数值列（如好评量、参评量）
- 可选的时间列（用于动态分析）
